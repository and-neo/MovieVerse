# MovieVerse

MovieVerse is a full-stack web application for discovering movies and TV shows, creating reviews, and managing personal favorites.

The application is being developed as part of a Bachelor's dissertation in Computing using the MERN technology stack.

---

## Technologies

- React.js
- Node.js
- Express.js
- MongoDB
- Mongoose
- TMDB API
- JWT Authentication
